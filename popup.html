사용법

  1. 웹페이지에서 일본어나 영어 텍스트를 드래그 선택
  2. 자동으로 나타나는 팝업에서 번역 결과 확인
  3. 필요시 "Anki에 저장" 버튼 클릭

Anki 저장 시 주의사항

  • Anki가 실행 중이어야 합니다
  • Anki Connect 애드온이 설치되어 있어야 합니다

Google 검색 자동 번역 사용법

  • eBook 앱에서 단어를 검색합니다.
  • 자동 번역이 활성화되어 있으면 검색어가 번역기에 자동 입력됩니다.
  • 복사/붙여넣기 없이 한국어 뜻을 바로 확인할 수 있습니다.
